# Dedede, Bandana Dee y Kirby

## Descripción

Este proyecto es un juego inspirado en Piedra, Papel o Tijera, pero utilizando personajes del universo de Kirby: Dedede, Bandana Dee y Kirby.

El jugador selecciona un personaje y la computadora elige otro al azar. El objetivo es conseguir tres victorias antes de perder tres veces.

## Reglas del juego

* Dedede vence a Kirby.
* Kirby vence a Bandana Dee.
* Bandana Dee vence a Dedede.
* Si ambos jugadores eligen el mismo personaje, se produce un empate.

## Cómo jugar

1. Abre el archivo `index.html` en un navegador.

[Aplicaciones](https://chatgpt.com/apps)

1. Haz clic en uno de los botones: Dedede, Bandana Dee o Kirby.
2. La computadora elegirá un personaje al azar.
3. El juego mostrará el resultado de la ronda y la puntuación.
4. Si consigues tres victorias, ganas la partida. Si pierdes tres veces, deberás intentarlo de nuevo.

## Tecnologías utilizadas

* HTML5
* CSS3
* JavaScript

## Uso de Inteligencia Artificial

* **ChatGPT:** se utilizó para comprender cómo implementar los eventos de los botones, la lógica del juego, el sistema de puntuación y para resolver dudas relacionadas con HTML, CSS y JavaScript.
